## DegenToken

The  Degen token allows Farcaster community members to earn reputation tokens.

### TOKEN_NAME

```solidity
string TOKEN_NAME
```

EIP-20 token name for this token.

### TOKEN_SYMBOL

```solidity
string TOKEN_SYMBOL
```

EIP-20 token symbol for this token.

### TOKEN_SUPPLY

```solidity
uint256 TOKEN_SUPPLY
```

Total number of tokens in circulation.
